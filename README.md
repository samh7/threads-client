# ThreadsClient

A simple front-end for a threads-like messaging app (single user, for now).

<img  src="./threads-app.gif"/>

## Features

1. Secure Cookies Jwt Auth
2. Route guards
3. Comment threading
4. Likes (Coming soon)

## Development server

To start a local development server, run:

```bash
pnpm i 
ng s -o
```
